## Como Rodar o aplicativo:

### 1 - Criar um banco de dados no Mysql chamado candidato-vagas com a collation utf8mb4_unicode_ci

### 2 - No terminal, executar o comando php artisan migrate

### 3 - Executar o comando php artisan db:seed --class=AdminSeed

### 4 - Executar o comando php artisan db:seed --class=VagasSeed

### 5 - Executar o comando php artisan db:seed --class=CandidatosSeed

### 5 - Executar o comando php artisan make:seeder CandidaturasSeed
